# Task Week 12: Depression Survey Analysis

This repository contains the data and analysis for the "Lefy Bena Ya Data" team's project exploring depression-related survey responses.

## Project Overview

The goal of this project is to analyze survey data related to depression to uncover insights, trends, and patterns. The analysis is documented in a Jupyter Notebook, detailing the data cleaning, exploration, and visualization processes. The final findings are summarized in a comprehensive PDF report.

## Repository Contents

The project files are located in the `Lefy Bena Ya Data/` directory:

- **`depression_analysis.ipynb`**: The main Jupyter Notebook containing the Python code for data processing, Exploratory Data Analysis (EDA), and data visualizations.
- **`depression_survey_clean.csv`**: The cleaned and preprocessed dataset used for the analysis.
- **`Contact Information (Responses) - Form Responses 1.csv`**: The raw survey responses collected via forms.
- **`Team Lefy Bena Ya Data Report.pdf`**: The final compiled report summarizing the methodology, findings, and conclusions of the analysis.

## Getting Started

To view or rerun the analysis locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/NaderFayed/task-week-12.git
   ```
2. Navigate to the project directory:
   ```bash
   cd task-week-12/Lefy\ Bena\ Ya\ Data/
   ```
3. Ensure you have Python and Jupyter Notebook installed. You will likely need common data science libraries such as `pandas`, `matplotlib`, and `seaborn`.
4. Open the notebook to explore the code:
   ```bash
   jupyter notebook depression_analysis.ipynb
   ```
